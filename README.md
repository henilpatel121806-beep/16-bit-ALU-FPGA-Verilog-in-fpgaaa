# 16-bit ALU Design using Verilog on FPGA

## Overview

This project implements a 16-bit Arithmetic Logic Unit (ALU) using Verilog HDL. The design supports arithmetic, logical, and shift operations along with status flags.

## Features

* 16-bit operations (ADD, SUB, AND, OR, XOR, etc.)
* Status Flags: CF, ZF, SF, OF, PF, AF
* Simulation using ModelSim
* FPGA implementation using Quartus

## Tools Used

* Verilog HDL
* Quartus
* ModelSim

## Results

* Successfully simulated all operations
* Verified hardware implementation using FPGA switches and LEDs

## Project Structure

* src/ → Verilog source files
* testbench/ → Simulation files
* docs/ → Report and screenshots
